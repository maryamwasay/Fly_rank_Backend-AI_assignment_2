# Student API - Flask + PostgreSQL + Docker

## Project Overview

This project is a Flask-based REST API for managing student records.

The main goal of this project was to replace temporary in-memory storage with a real PostgreSQL database running inside Docker while keeping the service and route layers unchanged.

The project demonstrates:
- Flask REST API development
- Layered architecture
- Repository pattern
- PostgreSQL database integration
- SQLAlchemy ORM
- Docker and Docker Compose
- Persistent database storage using Docker volumes
