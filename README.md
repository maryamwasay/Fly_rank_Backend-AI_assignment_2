# Backend_AI_Engineering_Assignment_2
# Student API - Flask + PostgreSQL + Docker

## Overview

This project is a Student Management REST API built using Flask.

The purpose of this project is to demonstrate how a backend application can move from temporary in-memory storage to a real PostgreSQL database while keeping the application architecture unchanged.

The project uses Docker and Docker Compose to run the Flask application and PostgreSQL database together as a complete local development environment.

## Features

- Create student records
- Retrieve all student records
- Delete student records
- PostgreSQL database integration
- SQLAlchemy ORM for database operations
- Dockerized Flask application
- Persistent database storage using Docker volumes
- Layered backend architecture


## Technologies Used

- Python 3.12
- Flask
- SQLAlchemy
- PostgreSQL 16
- Docker
- Docker Compose
- Postman
